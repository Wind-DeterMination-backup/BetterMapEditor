# Changelog

## v1.0.1
- 🛠️ 修复 `Generate` 预览线程中 `DraggableMirrorFilter` 的跨类加载器访问问题，避免黑色遮罩卡住与预览丢失。
- 🎯 保持 `Mirror` 原版本地化名称，并兼容角度/旋转参数。
- 🧲 修复镜像轴逻辑访问链，恢复预览中的可拖拽对称轴与实时镜像更新。

## v1.0.0
- 🚀 首发 BetterMapEditor：支持在地图编辑器自动生成预览里拖动镜像对称轴。
- 🧭 新增可视化轴心点，拖动时实时刷新预览结果。
- 🔁 自动兼容原版 `MirrorFilter`：保留角度/旋转参数并无缝替换为可拖拽版本。
- 📦 完成发布流水线：tag 触发构建 `.zip` / `.jar` / `-android.jar` 并自动发布 Release（含更新日志）。
